# 🔐 Security Policy

## 📌 Versões Suportadas

Atualmente, este projeto está em fase de desenvolvimento ativo. Apenas a versão mais recente recebe atualizações e correções de segurança.

| Versão | Suporte         |
| ------ | --------------- |
| 1.0.x  | ✅ Ativo         |
| < 1.0  | ❌ Não suportado |

---

## 🚨 Reportar Vulnerabilidades

Se você identificar alguma vulnerabilidade ou falha de segurança, por favor:

📩 Entre em contato através de:

* GitHub Issues (preferencialmente privadas, se possível)
* Ou diretamente pelo GitHub: https://github.com/Ruan-Marcelo

---

## ⏱️ Tempo de Resposta

* Confirmação inicial: até **48 horas**
* Análise completa: até **5 dias úteis**

---

## 🔍 Processo

Após o envio:

1. A vulnerabilidade será analisada
2. Se confirmada, será corrigida o mais rápido possível
3. Um update será publicado no repositório
4. O crédito será dado ao responsável pelo reporte (se desejado)

---

## ⚠️ Boas Práticas

Para manter a segurança da aplicação:

* Utilize apenas arquivos de imagem válidos (`.jpg`, `.jpeg`, `.png`)
* Evite inserir dados sensíveis no sistema
* Mantenha o ambiente e dependências sempre atualizados

---

## 🛡️ Observação

Este projeto é de caráter educacional e pode não cobrir todos os cenários de segurança de uma aplicação em produção.
